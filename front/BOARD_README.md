# 게시판 기능 구현 완료

## 📋 개요

머니빈 프로젝트에 3가지 유형의 게시판을 구현했습니다:
- **자유게시판** (free): 모든 로그인 사용자가 글 작성 및 댓글 가능
- **금융기사** (news): 크롤링된 뉴스 표시, 읽기 전용
- **금융정보** (info): 관리자만 작성 가능, 모든 로그인 사용자는 댓글 가능

## 🎨 디자인

기존 프로젝트의 스타일과 일관성을 유지했습니다:
- 머니빈 브랜드 컬러 (#00a651) 사용
- 기존 페이지와 동일한 레이아웃 구조
- 반응형 디자인 적용

| 경로 | 설명 |
|------|------|

## 🎯 주요 기능

### 1. 게시글 목록 (BoardListView)
- ✅ 게시판 타입별 게시글 목록 표시
- ✅ 검색 기능 (제목, 내용, 작성자)
- ✅ 페이지네이션 (한 페이지당 10개)
- ✅ 공지사항 배지 표시
- ✅ 댓글 수 표시
- ✅ 조회수, 작성일 표시
- ✅ 권한에 따른 글쓰기 버튼 표시/숨김

### 2. 게시글 상세 (BoardDetailView)
- ✅ 게시글 상세 내용 표시
- ✅ 작성자에게만 수정/삭제 버튼 표시
- ✅ 댓글 목록 및 작성
- ✅ 댓글 작성자에게만 삭제 버튼 표시
- ✅ 로그인 필수 안내 (미로그인 시)

### 3. 게시글 작성/수정 (BoardWriteView)
- ✅ 제목/내용 입력 폼
- ✅ 글자 수 카운터
- ✅ 관리자에게만 공지사항 체크박스 표시
- ✅ 작성 권한 검증
- ✅ 수정 시 기존 데이터 로드

## 🔐 권한 관리

### 자유게시판 (free)
- **글쓰기**: 로그인 사용자
- **댓글**: 로그인 사용자
- **수정/삭제**: 본인만 가능

### 금융기사 (news)
- **글쓰기**: 불가능 (크롤링 데이터)
- **댓글**: 불가능 (읽기 전용)

### 금융정보 (info)
- **글쓰기**: 관리자만 가능 (`userRole === 'admin'`)
- **댓글**: 로그인 사용자
- **수정/삭제**: 관리자만 가능

## 📡 API 연동 준비

### 백엔드에서 구현해야 할 API 엔드포인트:

#### 게시글 API
```
GET    /api/board/{type}/           # 게시글 목록 조회
POST   /api/board/{type}/           # 게시글 작성
GET    /api/board/{type}/{id}/      # 게시글 상세 조회
PUT    /api/board/{type}/{id}/      # 게시글 수정
DELETE /api/board/{type}/{id}/      # 게시글 삭제
```

#### 댓글 API
```
GET    /api/board/comments/{postId}/    # 댓글 목록 조회
POST   /api/board/comments/{postId}/    # 댓글 작성
DELETE /api/board/comments/{id}/        # 댓글 삭제
```

## 🚀 사용 방법

### 네비게이션 바에서 접근
```
게시판 > 자유게시판
게시판 > 금융정보(꿀팁)
게시판 > 금융기사
```

### 메인 페이지에서 접근
- 금융기사 섹션의 "더보기" 클릭
- 금융정보 섹션의 "더보기" 클릭

### 프로그래밍 방식 접근
```javascript
// 자유게시판으로 이동
router.push({ name: 'board-list', params: { type: 'free' } })

// 게시글 상세보기
router.push({ name: 'board-detail', params: { type: 'free', id: 1 } })

// 게시글 작성
router.push({ name: 'board-write', params: { type: 'free' } })
```
